# Morning Briefing

Generate the Monday Morning CEO Briefing for the current week.

## Instructions

You are the AI Employee generating a proactive executive briefing.

1. **Read** `AI_Employee_Vault/Business_Goals.md` — note revenue targets and active projects.

2. **Read** `AI_Employee_Vault/Accounting/` folder — look for any transaction logs or CSV files.

3. **Scan** `AI_Employee_Vault/Done/` — list all tasks completed this week (filter by date).

4. **Scan** `AI_Employee_Vault/Plans/` — list any plans created this week.

5. **Scan** `AI_Employee_Vault/Logs/` — read all log files from the past 7 days and aggregate:
   - Total actions taken
   - Any errors or failures
   - Unusual activity

6. **Generate** the briefing file at:
   `AI_Employee_Vault/Briefings/<YYYY-MM-DD>_Monday_Briefing.md`

   Use this structure:
   ```
   ---
   generated: <ISO timestamp>
   period: <start_date> to <end_date>
   ---

   # Monday Morning CEO Briefing

   ## Executive Summary
   <2-3 sentence summary of the week>

   ## Revenue
   - This Week: $X
   - MTD: $X (X% of $Y target)
   - Trend: On track / Behind / Ahead

   ## Completed Tasks
   - [x] Task 1
   - [x] Task 2

   ## Bottlenecks
   | Task | Expected | Actual | Delay |
   |------|----------|--------|-------|

   ## Proactive Suggestions
   ### Cost Optimization
   - List any subscriptions or costs to review

   ### Upcoming Deadlines
   - List from Business_Goals.md active projects

   ---
   *Generated by: AI Employee v0.1*
   ```

7. **Update Dashboard.md** — append a link to the new briefing in Recent Activity.

## Notes

- If accounting data is unavailable, note "Data pending — drop bank CSV into /Inbox/"
- If no tasks were completed, state so clearly — do not fabricate data
- Suggest 1-3 actionable items based on what you observed

## Output

Confirm the briefing file path and provide a 3-bullet executive summary.
